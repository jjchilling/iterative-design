Introduction & Context

From our sketches, we decided to incorporate any elements we felt best streamlined the user experience in order to project a casual atmosphere. Notably, we wanted the focal point of the home page to be the New Event button and the New Hangout interfaces.


To simplify the New Hangout user flow in particular, we decided to represent Hangouts with a calendar-adjacent interface to directly exhibit times in which all guests are available. Furthermore, we reduced the New Hangout creation page to only the necessary inputs.


On a separate note, while familiarizing ourselves with the Partiful interface, we found a few features we really liked that we thought could benefit from greater discoverability, such as Party Genie and Google Calendar Sync. We thus emphasized them on our homepage by isolating them within their own features (e.g. in buttons or next to toggles).
